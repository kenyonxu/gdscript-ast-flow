架构推导

解析器需要两个输出层

.gd 源码
　│
　├─ GDScriptTokenizer (词法分析)
　│　　生成: Token 流
　│
　├─ GDScriptASTParser (语法分析)
　│　　生成: AST (ClassNode, FunctionNode, ...)
　│
　└─ GDScriptSymbolResolver (符号解析) ← 新增
　　　　遍历 AST, 构建:
　　　　• SymbolTable (变量/函数/信号 → 声明位置)
　　　　• CallGraph (调用关系图)
　　　　• DefUseChain (定义-使用链)
　　　　• SignalGraph (信号连接图)
